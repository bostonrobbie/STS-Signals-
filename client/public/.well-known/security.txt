# Responsible disclosure
#
# We welcome reports of security vulnerabilities. Please email
# support@stsfutures.com with:
#   1. A clear description of the vulnerability
#   2. Steps to reproduce
#   3. Impact assessment (what an attacker could do)
#   4. Any suggested mitigation
#
# We aim to acknowledge initial reports within 72 hours and to
# provide a fix or detailed remediation plan within 30 days for
# valid issues. Critical issues (e.g. affecting subscriber data
# or signal delivery) are prioritized.
#
# Out of scope:
#   - Volumetric DDoS attacks
#   - Social engineering of staff or users
#   - Issues already publicly disclosed
#   - Vulnerabilities requiring physical access to the user's device
#
# We do not currently run a paid bounty program but acknowledge
# valid reports publicly (with reporter consent) on the
# acknowledgments page.
